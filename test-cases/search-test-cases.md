# Тест-кейсы: поиск товара

## TC-SEARCH-01. Поиск существующего товара

**Приоритет:** High  
**Тип:** Positive

### Тестовые данные

Поисковый запрос: laptop

### Шаги

1. Открыть главную страницу магазина.
2. Нажать на поле поиска.
3. Ввести запрос laptop.
4. Нажать кнопку поиска.

### Ожидаемый результат

Открывается страница с результатами поиска. Отображаются товары по запросу laptop.

## TC-SEARCH-02. Поиск несуществующего товара

**Приоритет:** Medium  
**Тип:** Negative

### Тестовые данные

Поисковый запрос: qwerty12345

### Шаги

1. Открыть главную страницу.
2. Ввести qwerty12345 в поле поиска.
3. Нажать кнопку поиска.

### Ожидаемый результат

Отображается сообщение "Товары не найдены".

## TC-SEARCH-03. Поиск с пустым полем

**Приоритет:** Medium  
**Тип:** Negative

### Шаги

1. Открыть главную страницу.
2. Оставить поле поиска пустым.
3. Нажать кнопку поиска.

### Ожидаемый результат

Поиск не выполняется или появляется сообщение о необходимости ввести запрос.

## TC-SEARCH-04. Поиск со спецсимволами

**Приоритет:** Low  
**Тип:** Negative

### Тестовые данные

Поисковый запрос: @#$%^&*

### Шаги

1. Открыть главную страницу.
2. Ввести спецсимволы в поле поиска.
3. Нажать кнопку поиска.

### Ожидаемый результат

Сайт не ломается и корректно обрабатывает запрос.
