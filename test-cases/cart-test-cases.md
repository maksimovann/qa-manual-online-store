# Тест-кейсы: корзина

## TC-CART-01. Добавление товара в корзину

**Приоритет:** High  
**Тип:** Positive

### Предусловия

Товар есть в наличии.

### Шаги

1. Открыть каталог товаров.
2. Открыть карточку товара.
3. Нажать кнопку "Добавить в корзину".
4. Открыть корзину.

### Ожидаемый результат

Товар отображается в корзине.

## TC-CART-02. Удаление товара из корзины

**Приоритет:** High  
**Тип:** Positive

### Предусловия

Товар уже добавлен в корзину.

### Шаги

1. Открыть корзину.
2. Нажать кнопку "Удалить" рядом с товаром.

### Ожидаемый результат

Товар удаляется из корзины.

## TC-CART-03. Изменение количества товара

**Приоритет:** High  
**Тип:** Positive

### Предусловия

Товар уже находится в корзине.

### Шаги

1. Открыть корзину.
2. Нажать кнопку "+" рядом с товаром.

### Ожидаемый результат

Количество товара увеличивается, итоговая сумма пересчитывается.

## TC-CART-04. Проверка итоговой суммы

**Приоритет:** High  
**Тип:** Positive

### Предусловия

В корзину добавлено несколько товаров.

### Шаги

1. Открыть корзину.
2. Проверить цены товаров.
3. Проверить итоговую сумму.

### Ожидаемый результат

Итоговая сумма рассчитана корректно.

## TC-CART-05. Переход к оформлению заказа

**Приоритет:** Medium  
**Тип:** Positive

### Предусловия

В корзине есть хотя бы один товар.

### Шаги

1. Открыть корзину.
2. Нажать кнопку "Оформить заказ".

### Ожидаемый результат

Открывается страница оформления заказа.
