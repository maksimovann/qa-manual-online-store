# Тест-кейсы: оформление заказа

## TC-CHECKOUT-01. Успешное оформление заказа

**Приоритет:** High  
**Тип:** Positive

### Предусловия

Товар добавлен в корзину.

### Тестовые данные

- Имя: Анна
- Телефон: +79990000000
- Адрес: Москва, Тестовая улица, 1

### Шаги

1. Открыть корзину.
2. Нажать кнопку "Оформить заказ".
3. Заполнить обязательные поля.
4. Выбрать способ доставки.
5. Нажать кнопку "Подтвердить заказ".

### Ожидаемый результат

Заказ успешно оформлен.

## TC-CHECKOUT-02. Оформление заказа с пустыми полями

**Приоритет:** High  
**Тип:** Negative

### Предусловия

Товар находится в корзине.

### Шаги

1. Открыть оформление заказа.
2. Оставить обязательные поля пустыми.
3. Нажать кнопку "Подтвердить заказ".

### Ожидаемый результат

Заказ не оформляется. Появляются сообщения об ошибке.

## TC-CHECKOUT-03. Некорректный номер телефона

**Приоритет:** Medium  
**Тип:** Negative

### Тестовые данные

Телефон: abcdefg

### Шаги

1. Открыть страницу оформления заказа.
2. Заполнить обязательные поля.
3. Ввести некорректный номер телефона.
4. Нажать кнопку "Подтвердить заказ".

### Ожидаемый результат

Заказ не оформляется. Под полем телефона появляется ошибка.

## TC-CHECKOUT-04. Оформление заказа без выбора доставки

**Приоритет:** Medium  
**Тип:** Negative

### Предусловия

Товар находится в корзине.

### Шаги

1. Открыть страницу оформления заказа.
2. Заполнить обязательные поля.
3. Не выбирать способ доставки.
4. Нажать кнопку "Подтвердить заказ".

### Ожидаемый результат

Заказ не оформляется. Пользователь видит сообщение о необходимости выбрать доставку.
