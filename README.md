# QA Manual Testing Project: Online Store

## О проекте

Это учебный проект по ручному тестированию интернет-магазина.

Я взяла типовой сценарий онлайн-магазина и подготовила для него базовую тестовую документацию: тест-план, чек-листы, тест-кейсы и баг-репорты.

Проект нужен для практики и для портфолио junior QA.

## Что проверяется

В проекте проверяются основные функции интернет-магазина:

- авторизация
- поиск товара
- карточка товара
- корзина
- оформление заказа
- негативные сценарии

## Что есть в репозитории

```text

test-cases/       тест-кейсы по основным разделам
bug-reports/      примеры баг-репортов
